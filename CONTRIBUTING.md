# Contributing

Thank you for contributing to GT8004 SDK!

## Quick Start

1. Fork & clone
2. `pip install -e .[dev]`
3. Make changes
4. `pytest tests/ -v`
5. Open PR

## Commit Style

Use [Conventional Commits](https://www.conventionalcommits.org/):
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `test:` Tests

Example: `feat: add batch size configuration`

## Questions?

Open an issue or discussion!
